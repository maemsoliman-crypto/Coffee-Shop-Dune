# Dune Coffee Website

This project is a simple coffee shop website created as part of a Level 2 web development course. The website represents a luxury themed coffee shop called **Dune Coffee** and showcases a homepage along with menu pages for drinks and pastries.

The aim of the project was to design and develop a small multi-page website using HTML and CSS while following a client brief and wireframe design.

## Live Website

The website is hosted using GitHub Pages.
[View the website](https://maemsoliman-crypto.github.io/Coffee-Shop-Dune/)
## Pages

The website contains three main pages:

* **Home Page** – Introduces the coffee shop and provides a welcoming overview.
* **Drinks Menu** – Displays a selection of speciality drinks with images and prices.
* **Pastries Menu** – Shows a range of pastries available at the coffee shop.

Each page contains a consistent navigation bar and footer to keep the layout uniform across the website.

## Technologies Used

The project was built using the following technologies:

* **HTML** – Used to create the structure of the webpages.
* **CSS** – Used to style the website including layout, colours and typography.
* **Google Fonts** – Used to add custom fonts for a more premium visual style.
* **Visual Studio Code** – Used as the main development environment.
* **Live Server Extension** – Used to preview and test the website during development.

## Features

Some of the main features of the website include:

* Responsive layout that adapts to different screen sizes
* Fixed navigation bar for easier navigation
* Image based menu cards for drinks and pastries
* Consistent colour palette to match the luxury coffee shop theme
* Simple custom SVG logo used in the navigation bar
* Accessible images using alt text

## Design

The design of the website follows a **luxury coffee shop theme** using a burgundy and gold inspired colour palette. The layout was developed using wireframes provided in the project brief to ensure the structure matched the client requirements.

## Testing

The website was tested during development using the Live Server extension in Visual Studio Code. It was also tested across different browsers and screen sizes to ensure the layout remained consistent and responsive. Accessibility testing was carried out using Windows Narrator to check that images and content could be interpreted by screen readers.
